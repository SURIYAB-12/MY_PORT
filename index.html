Power Optimized AES Using CFA for Secure Memory Access and FPGA Implementation

Ensuring secure boot and protected memory access is essential for embedded and IoT systems operating in safety- and security-critical domains. However, software-based cryptographic mechanisms often introduce significant performance overheads and remain vulnerable to tampering. This paper presents a hardware-accelerated framework for secure boot and memory access control using the Advanced Encryption Standard (AES) implemented on a Field Programmable Gate Array (FPGA). The proposed design integrates an optimized AES core with on-chip memory controllers to authenticate firmware at startup and enforce controlled access to sensitive memory regions. To enhance efficiency, parallelization and pipelining techniques are employed to reduce latency and improve hardware utilization. Implementation on a Xilinx FPGA demonstrates a 1.67× reduction in power consumption and reduced boot-time exposure to unauthorized code compared to conventional software-based approaches. The proposed architecture enhances system resilience against tampering and replay attacks while offering a low-power, scalable solution for real-time security in embedded and IoT applications.
